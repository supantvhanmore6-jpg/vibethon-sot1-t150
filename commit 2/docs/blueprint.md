# **App Name**: AIML Quest

## Core Features:

- Interactive Learning Path: A structured curriculum across Beginner, Intermediate, and Advanced levels, with each topic featuring explanations, real-life examples, illustrations, and interactive demos or quizzes.
- Coding Playground: An embedded code editor with an output panel for pre-made coding challenges in Python-style pseudocode and a drag-and-drop ML pipeline builder.
- Gamified Mini-Labs: A dashboard of interactive mini-games that teach AI/ML concepts through gameplay, featuring XP points, coins, achievement badges, and animated rewards.
- Real-World AI Simulations: Interactive simulations demonstrating practical applications of AI across various domains, allowing users to manipulate inputs and observe AI responses.
- Quiz Arena: A hub for topic-wise and timed quizzes with diverse question types, score tracking, answer feedback, suggested revisions, and a leaderboard.
- Personalized Dashboard: A user-specific page displaying progress, earned badges, quiz performance, daily streaks, and personalized recommendations for the next learning steps.
- AI Chatbot Assistant: An intelligent tool providing real-time support, answering questions about AIML concepts, offering coding assistance, and explaining complex topics.

## Style Guidelines:

- Primary interactive elements color: A vibrant cyan, #22D3EE (HSL(187, 85%, 53%)), chosen to create striking neon glowing effects against the dark background.
- Background color: A deep, rich blue-purple, #0B1026 (HSL(228, 47%, 10%)), providing a dark, futuristic canvas that allows interactive elements to pop.
- Accent color: A brilliant violet, #7C3AED (HSL(264, 80%, 58%)), used for secondary highlights, subtle glow effects, and brand elements, complementing the primary cyan.
- Text color: Pure white (#FFFFFF) for maximum contrast and readability against the dark theme.
- Headline font: 'Orbitron' (geometric sans-serif) for its distinctly futuristic and tech-inspired aesthetic.
- Body text font: 'Inter' (grotesque-style sans-serif) for clean readability and a modern, professional appearance.
- Code snippets font: 'Source Code Pro' (monospaced sans-serif) for clarity and authenticity within the coding playground.
- Sleek, futuristic vector icons representing AI, coding, data, and learning, with subtle glow effects and interactive animations on hover.
- Glassmorphism design for cards and UI elements, with rounded corners, lending a sophisticated, transparent feel; responsive grid layouts optimized for desktop and mobile viewing.
- Dynamic background animations featuring animated particles and floating AI-related icons; smooth page transitions and hover effects; neon glowing buttons, and animated feedback for user actions like completing games or quizzes.
